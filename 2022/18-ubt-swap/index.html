Ubuntu 上如何配置 Swap(交换分区)

Swap是什么

Linux devides its physical RAM into chunks of memory called pages(页面). Swapping(动词"交换") is the process whereby a page of memory is copied to the preconfigured space on the hard disk called swap space(名词”交换空间“), to free up that page of memory.

The compined sizes of the physical memory and swap space is the amount of virtual memory available.

简单来说，swap就是在硬盘上分一块空间当做扩展内存来用，linux系统在内存不足时会把不活跃的一部分内存放到swap中，解决内存不足的情况。

通常树莓派以及一些轻量服务器的内存较小（可能只有1-2G），如果因为内存不足导致一些程序无法运行，可以通过配置swap来给内存“扩容”。
