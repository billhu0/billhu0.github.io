    为什么会导致无限递归？

    因为在外部访问 obj.a 相当于运行getter get() 函数得到函数返回值，但是在get() 内 return obj.a 也相当于访问了 obj.a, 也会运行 get() 函数，相当于 get() { return get(); }，从而导致套娃。
