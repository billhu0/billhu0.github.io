Lecture 6: Joint Distributions

Covariance

Definition

Cov(x,y)=E((X−EX)(Y−EY))=E(XY)−E(X)E(Y) \mathrm{Cov}(x,y) = E((X-EX)(Y-EY)) = E(XY) - E(X)E(Y) Cov(x,y)=E((X−EX)(Y−EY))=E(XY)−E(X)E(Y)

Properties

  • Cov(X,X)=Var(X)Cov(X,X) = Var(X)Cov(X,X)=Var(X)

  • Cov(X,Y)=Cov(Y,X)Cov(X,Y) = Cov(Y,X)Cov(X,Y)=Cov(Y,X)

  • Cov(X,c)=0Cov(X,c) = 0Cov(X,c)=0

  • Cov(aX,Y)=a⋅Cov(X,Y)Cov(aX, Y) = a \cdot Cov(X,Y)Cov(aX,Y)=a⋅Cov(X,Y)

  • Cov(X+Y,Z)=Cov(X,Z)+Cov(Y,Z)Cov(X+Y,Z) = Cov(X,Z)+ Cov(Y,Z)Cov(X+Y,Z)=Cov(X,Z)+Cov(Y,Z)

  • Cov(X+Y,Z+W)=Cov(X,Z)+Cov(X,W)+Cov(Y,Z)+Cov(Y,W)Cov(X+Y, Z+W) = Cov(X,Z) + Cov(X,W) + Cov(Y,Z) + Cov(Y,W)Cov(X+Y,Z+W)=Cov(X,Z)+Cov(X,W)+Cov(Y,Z)+Cov(Y,W)

  • Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)Var(X+Y) = Var(X) + Var(Y) + 2Cov(X,Y)Var(X+Y)=Var(X)+Var(Y)+2Cov(X,Y)

  • For nnn r.v.s X1,⋯ ,XnX_1, \cdots, X_nX1​,⋯,Xn​,

    Var(X1+⋯+Xn)=Var(X1)+⋯+Var(Xn)+2∑i<jCov(Xi,Yj)Var(X_1 + \cdots + X_n) = Var(X_1) + \cdots + Var(X_n) + 2 \sum_{i<j}Cov(X_i, Y_j) Var(X1​+⋯+Xn​)=Var(X1​)+⋯+Var(Xn​)+2i<j∑​Cov(Xi​,Yj​)

Correlation

Definition

Corr(X,Y)=Cov(X,Y)Var(X)Var(Y) Corr(X,Y) = \frac{Cov(X,Y)}{\sqrt{Var(X)Var(Y)}} Corr(X,Y)=Var(X)Var(Y)​Cov(X,Y)​

Cov(X,Y)=0   or   Corr(X,Y)=0   ⇒   Uncorrelated Cov(X,Y) = 0 \mathrm{\ \ \ or\ \ \ } Corr(X,Y) = 0 \ \ \ \Rightarrow \ \ \ \mathrm{Uncorrelated} Cov(X,Y)=0   or   Corr(X,Y)=0   ⇒   Uncorrelated
